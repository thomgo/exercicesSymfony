# Correction exercices sur Symfony4

Ce projet est une correction simple d'exercices sur Symfony4 proposée à mes étudiants, dans le cadre de mon poste de formateur en développement Web à l'ADEP de Roubaix.

Ces exercices sont donnés en semaine de découverte sur Symfony et vise à réaliser un blog élémentaire et ainsi découvrir les outils fondamentaux du framework.
